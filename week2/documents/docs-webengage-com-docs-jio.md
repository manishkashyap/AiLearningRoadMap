Follow the below steps to configure Jio as your RCS Service Provider (RSP).
As shown above
Step 1: Select RSP
Select
Jio
from the List of Available RSPs. In doing so, you will be prompted by a configuration modal.
Step 2: Name your Configuration
Please add a user-friendly name that enables you to identify the right RSP for a campaign while
creating it
. This comes in handy especially, when you have multiple accounts with the same RSP for sending different types of campaigns.
Step 3: Add API Key
Navigate to your
Jio
to find and copy your API Key. Paste it in the configuration modal to connect your account with WebEngage.
Step 4: Add Custom headers
Example: If the account is in the US environment, click “Add Header” and specify:
env
: USA
dsn_endpoint
: as per the respective environment.
Step 5: Add RSP
Click Add RSP, and you're good to go
👍
Congratulations!
You've now successfully integrated Jio with your WebEngage dashboard. You can test the integration by directly creating a RCS template from WebEngage; there’s no need to create the RCS template on the
Jio dashboard.
