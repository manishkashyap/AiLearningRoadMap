As shown above:
Step 1: Select ESP
Select
Mailercloud
from the List of Available ESPs. In doing so, you will be prompted by a configuration modal.
Step 2: Name your Configuration
Please add a user-friendly name that enables you to identify the right ESP for a campaign
while creating it
. This comes in handy especially when you have multiple accounts with the same ESP for sending different types of campaigns
(Transactional/ Promotional).
For example, if you have configured a
Mailercloud
account for sending transactional campaigns, then you can name the configuration:
Mailercloud T
Mailercloud Transactional
Step 3: Add Domain
Please add the same domain that you have verified for email marketing in your
Mailercloud dashboard.
Step 4: Add API Key
Refer to the complete
Mailercloud API Key Generation Guide
for detailed instructions on generating your API Key. Once your API Key is generated, copy it and paste the API Key into the configuration modal.
Step 5: Add ESP
Click on
Add ESP
and you're good to go!
