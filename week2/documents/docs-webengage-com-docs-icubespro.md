Step 1: Select ESP
Select
iCubesPro
from the List of Available ESPs. In doing so, you will be prompted by a configuration modal.
Step 2: Name your Configuration
Please add a user-friendly name that enables you to identify the right ESP for a campaign
while creating it
. This comes in handy especially when you have multiple accounts with the same ESP for sending different types of campaigns
(Transactional/ Promotional).
For example, if you have configured an
iCubesPro
account for sending promotional campaigns, then you can name the configuration:
iCubes Promo
iCubesPro P
Step 3: Add API Key
You can choose to use an existing
API Key
by proving suitable permissions to WebEngage (in your
iCubesPro dashboard
). Or, you can create a new API Key to connect the ESP to your WebEngage account.
Step 4: Add Custom Headers
Please get in touch with your relationship manager at iCubesPro if these authentication headers aren't readily available in your dashboard.
Step 5: Add ESP
Click
Add ESP
, and you're good to go!
