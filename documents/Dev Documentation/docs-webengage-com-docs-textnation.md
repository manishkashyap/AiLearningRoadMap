Channel Configuration
SMS
TextNation
Follow the below steps to configure
TextNation
as your SMS Service Provider (SSP).
Click to enlarge
As shown above:
Step 1: Select SSP
Select
TextNation
from the List of Available SSPs. In doing so, you will be prompted by a configuration modal.
Step 2: Name your Configuration
Please add a user-friendly name that enables you to identify the right SSP for a campaign
while creating it
. This comes in handy especially, when you have multiple accounts with the same SSP for sending different types of campaigns.
For example, if you have chosen to use
TextNation
for sending both
Transactional and Promotional campaigns
, then we recommend that you indicate the same by naming the configuration something like:
TextNation Global a/c
TextNation (all campaigns)
Step 3: Add API Key
Navigate to your
TextNation
to find and copy your API Key. Paste it in the configuration modal to connect your account with WebEngage.
Step 4: Add Principal Entity ID
(Applicable only if you are sending messages to users located in India via domestic messaging pipeline. International messaging routes remain unaffected.)
As per
TRAI's latest SMS Marketing regulations for India,
you must be registered on a DLT portal as a business entity. Once registered, a unique
Principle Entity ID (PE ID)
will be assigned to you.
W.e.f from Nov 19, 2020, it is mandatory for all SSPs to pass a PE ID in the payload of each SMS message being sent to Indian phone numbers. We have partnered with MSG91 to make this a seamless process for you.
Simply add your
PE ID
during configuration - we will ensure that it is relayed to the SSP for each SMS sent to your users.
Step 5: Add SSP
Click
Add SSP
, and you're good to go!*
