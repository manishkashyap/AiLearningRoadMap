Follow the below steps to configure
CM
as your
SMS Service Provider (SSP)
.
Click to enlarge
As shown above:
Step 1: Select SSP
Select CM from the List of Available SSPs. In doing so, you will be prompted by a configuration modal.
Step 2: Name your Configuration
Please add a user-friendly name that enables you to identify the right SSP for a campaign
while creating it
. This comes in handy especially, when you have multiple accounts with the same SSP for sending different types of campaigns.
For example, if you have chosen to use
CM
for sending
Promotional campaigns,
then we recommend that you indicate the same by naming the configuration something like:
CM Promo
CM Mobile Promo Pipeline
Step 3: Add API Key
Navigate to your
CM
dashboard to find your
API Key
. Paste them in the configuration modal to connect your account with WebEngage.
Step 4: Add Country Code (Optional)
📘
Please Note
This needs to be added only if you have not added the country code in the user's number. For example, if the Indian user's number is saved as '76348xxxxx' in your database instead of '+9176348xxxxx' then kindly enter the country code while configuring the
CM SSP
.
Kindly
do not
add the '+' sign with the
Country Code
while configuring
CM SSP
.
Add the telephone country code of the location to which you'll be sending the SMSes. This must be the same as the country selected while setting up your
CM SSP
account.
📘
As per CM Guidelines:
If the
Country Code
is not present with SSP credentials settings then there are high chances that the message delivery will fail.
Step 5: Add SSP
Click
Add SSP
, and you're good to go!
Step 6: Add WebEngage Webhook URL in CM Dashboard (Highly Recommended)
Adding the WebEngage Webhook in CM will enable us to receive delivery status notifications for each user. This includes campaign performance indicators like the
message delivered, failed, and queued
.
Here's how you can go about it:
Click to enlarge
Step 5.1.:
As shown below, you will find your CM integration under the section, Your
SMS Service Provider
List.
Step 5.2.:
Click the overflow menu placed on the extreme right, click
View Webhook URL
, and copy the link.
Step 5.3.:
Paste the
WebEngage Webhook URL
under the relevant field in your
CM dashboard
and click
Save
. Please connect with your
CM Account Manager
for further assistance.
👍
Congratulations!
You've now successfully integrated
CM
with your WebEngage dashboard.
You can test the integration by creating a test SMS campaign and sending it to a group of internal users (aka your teammates).
