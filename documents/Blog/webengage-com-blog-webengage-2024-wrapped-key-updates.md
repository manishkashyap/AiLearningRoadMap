WebEngage 2024 Wrapped: Key Updates You Need to Know!
2024 has been a year of major strides for WebEngage, with exciting new features and enhancements transforming how brands engage with their customers.
From launching new communication channels to enhancing analytics and customer data management, we’ve rolled out updates that make customer engagement easier and more impactful.
Together, let’s explore our standout features and enhancements of the year.
Communication Channels & Engagement
Rich Communication Services (RCS) Channel Launch
RCS allows you to send rich media (images, videos, buttons) and go beyong the 160-character SMS limit. Customize messages with branded profiles and enjoy secure, verified communication. With analytics like delivery and read receipts, gain better insights. Choose from Text, Rich Card, or Rich Carousel formats to suit your messaging needs.
WhatsApp: New Features & Enhancements
Dynamic Button Link Tracking:
Track user actions via dynamic buttons in templates, adding custom domains for tracking.
Carousel Template:
Showcase up to 10 products in a horizontal, customizable carousel. Choose from image or video options.
Enhanced Button Capabilities:
Add up to 10 buttons, including quick replies and call-to-action buttons.
Automated Template Approval:
Submit and track WhatsApp template approvals directly from WebEngage.
Media File Size Update:
Updated media limits: Images (5 MB), Videos (16 MB), Documents (100 MB).
Push Notifications
New Features:
Include custom sounds (iOS), large icons, auto-dismiss (Android), and auto-scrolling carousels.
Text Overlay Layout:
Customize notifications with full-image backgrounds, or collapsed versions (Android).
Improved CTAs:
Add buttons for URLs, sharing, calling, snoozing, tracking, or custom actions.
In-App
Spin the Wheel Template:
Engage users with interactive Spin the Wheel campaigns.
Scratch Card Template:
Gamify interactions with Scratch Cards for personalized offers.
NPS Template:
Gather feedback with NPS surveys and tailor follow-up strategies.
JavaScript Support:
Customize notifications with JavaScript for dynamic, interactive campaigns with a 10,000-character HTML limit.
Engagement Score
Measure user activity with precision! Assign weightage to events, and the system calculates engagement scores from 1-10 based on user activity. Key features include:
Customizable Weightage: Adjust event weightage, lookback periods, and refresh frequency.
Score Clustering: Identify clear trends in user engagement and act on them effectively.
Campaign Orchestration
Batched Relays
Send notifications in batches by grouping multiple business events and triggering them at set intervals. This ensures cohesive messaging, improves user experience, and reduces notification fatigue.
Catalog-based Relays
Automate campaigns using real-time inventory updates for events like “Back in Stock” or “Price Drop.” Deliver timely, targeted messages to drive higher conversions and keep your communication relevant.
Universal Control Group: Uplift
Compare the behavior of the target group of the campaign against a control group, to accurately measure the uplift in critical metrics such as conversions, revenue, and orders. Measure campaign effectiveness accurately for data-driven decision-making.
Customer Data Platform (CDP)
Data Management: Predefining Data Model
Easily manage schemas from the dashboard under
Data Management
, choose between two schema options:
Fixed Data Model:
This model strictly adheres to predefined events and attributes, ideal for industries like financial services where data consistency is critical.
Flexible Data Model:
This model accepts dynamic data, perfect for businesses that need flexibility as data requirements evolve.
Precompute KPI User Page
Track performance trends directly on the
Users page
with:
Week-over-Week (WoW) and Month-over-Month (MoM) insights for DAU, WAU, MAU, and channel reachability.
Interactive trendline charts and persistent viewing preferences for effortless analysis.
Custom Alerts 2.0
Set thresholds for metrics like DAU, WAU, or MAU and receive notifications on deviations, without opening the dashboard!
Prebuilt Templates:
Quick-start alerts for common metrics like DAU decline.
Enhanced UX:
Intuitive interface with reusable custom alerts to save time.
Catalogs & Recommendations
Fallback Recommendations
Ensure users see relevant suggestions even if the primary strategy doesn’t have enough items. Add up to 3 fallback recommendations (Relevant, Personalized or Static), in order of priority to ensure a seamless experience.
Custom Recommendation API
Deliver personalized recommendations at scale using WebEngage’s new API. Upload recommendations via CSV (up to 200 MB) and choose between Relevant, Personalized, and Static recommendations.
Integrations & Platform Enhancements
Mail by WebEngage launch
Now you can send emails directly from WebEngage without external ESPs. This feature includes:
Real-time performance tracking and revenue attribution.
Domain health monitoring and simplified billing for ease of use.
ESP: AMP Support for Self-Serve
WebEngage’s Self-Serve ESP now supports AMP, enabling interactive email content like forms and carousels. This feature integrates easily with popular ESPs like SparkPost, SendGrid, and AWS SES.
Event Export: Integrations
Export event-level data directly from WebEngage to Mixpanel, Amplitude, and even Amazon S3. Move beyond cohorts and enjoy detailed event exports, completing a two-way data sync that gives you complete control over your data flow.
Auto Add UTM Parameters
Save time by automatically adding UTM parameters to your campaign links.
Feed data directly into Google Analytics for unified performance tracking.
Track campaigns across multiple tools and channels.
Available for Email, SMS, Web Push, WhatsApp, On-Site Notifications, and Web In-line.
Email Templating Engine: New Template Library
The new Template Library consolidates all email templates, including drag-and-drop, AMP-enabled, and custom HTML designs, into one place. It streamlines email creation with an intuitive, code-free interface, ensuring a smooth and efficient design process.
Actionable Analytics
Event Analytics 2.0
Hourly Granularity:
Move beyond daily trends and analyze user events on an hourly basis, moving beyond daily trends.
Comparison Tooltip:
View data trends with a new tooltip, showing the direction and rate of change for improved user behavior analysis.
New UI
Enhanced Plotting:
Highlight and isolate data points with easier trend analysis using consistent X-axis and dynamic Y-axis scaling.
More Data:
Display up to 20,000 series for handling larger datasets and revealing deeper insights.
New Charts:
Compare multiple metrics side-by-side with the new metrics chart and pin important metrics for quick access.
Smarter Date Selection:
Select custom date ranges with ease to focus on specific time periods.
More Metrics:
Compare occurrences and sessions in one view and use new aggregates like sum, average, and median.
Funnel Analytics 2.0
UI Improvements:
Streamlined funnel UI with easy access to delete, duplicate, rename, and filter steps via a three-dot action button.
Rename Funnel Steps:
Customize funnel step names for clearer tracking, like renaming to “First-Time Purchase” or “Repeat Buyer.”
Completion Time and Split By Options:
Add “Completion Time” and “Split By” options for deeper funnel analysis, tracking time spent on steps and splitting data by metrics like demographics.
Cohorts: Enhancements
Segment Filter:
Segment cohorts based on unique user characteristics for more targeted analysis, understanding behaviors like decision-making time for repeat versus first-time buyers.
Export to Lists:
Export static and refreshing lists from cohort analysis to effectively track user behaviors over time and target retained or dropped-off users.
Paths: Enhancements
Split Events by Attributes:
Split events by attributes for insights into behaviors based on factors like location, technology, or UTM parameters.
Advanced Filters — Segment/List:
Apply advanced filters to focus on specific user segments or lists, providing deeper insights into the paths taken by targeted users.
Add/Remove Rows:
Adjust the number of rows in your Path analysis to explore more or fewer user paths, uncovering uncommon interactions or refining your focus.
RFM: Enhancements
New Filters Addition:
Apply advanced filters to segment user data by event attributes, user attributes, and segments.
Export to Refreshing Lists:
Export RFM data to create refreshing lists that automatically update based on selected criteria.
Wrapping Up
These innovations are just the beginning of what’s possible with WebEngage. With each enhancement, we aim to make your strategies more powerful, data-driven, and customer-focused.
