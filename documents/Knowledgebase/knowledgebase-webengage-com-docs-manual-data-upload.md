Upload Data
Upload your User and Events data manually on WebEngage
User and Events data are extremely helpful in creating Segments and Journeys to create a successful marketing campaign. WebEngage enables you to consolidate your offline data or your historic user data by uploading it manually. You can convert your user data in the excel sheet in the form of a CSV file and upload it.
Upload User Data
There can be times when you would want to upload your own user data. This can be done by converting your existing data into a CSV file format and uploading it. However, while converting your file to a CSV file, some
guidelines
must be followed to avoid any error or data discrepancy.
Upload Events Data
Events data upload can help when you want to know whether the data being tracked is correct or not or if it needs changes. If the events or any of the event attributes related to it needs changes, it can be done following
managing the events data
.
